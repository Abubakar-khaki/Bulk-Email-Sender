# ⚡ Quick Start

## Step 1: Get Gmail App Password
Go to: https://myaccount.google.com/apppasswords

## Step 2: Edit the code
- Add your email
- Add your app password
- Add people to send to
- Write your message

## Step 3: Run
```
python "bulik email sender.py"
```

## Step 4: Done! 🎉
Email sent to everyone on your list!
